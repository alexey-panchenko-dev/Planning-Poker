# Poker Planning — Client

Клиентская часть приложения для проведения покер-планирования.
Построена на базе **React 19**, **TypeScript** и сборщика **Vite**.

## Стек технологий

- **Core:** React 19, TypeScript, Vite
- **State & Data:** Zustand, TanStack React Query, Axios
- **Real-time:** Socket.io-client
- **UI & UX:** Tailwind CSS (v4), Lucide React, React Joyride
- **Drag and Drop:** @dnd-kit/core

---

## Инструкция по запуску

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск проекта

```bash
npm run dev
```
